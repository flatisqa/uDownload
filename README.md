# yD — Ultimate Media Downloader 🚀

**yD** — это быстрое, мощное и визуально красивое десктопное приложение для загрузки аудио и видео с поддержкой большинства медиаплатформ. Оно построено на базе легендарного `yt-dlp` и объединяет максимальные функциональные возможности с современным интерфейсом и настраиваемым UI (включая множество красивых тем).

---

## ✨ Ключевые возможности (Features)

- **Скачивание плейлистов и отдельных видео** в один клик.
- **Поддержка 4K/8K**: скачивайте видео в наилучшем качестве.
- **Интеллектуальная обрезка**: выбирайте нужный диапазон времени прямо перед скачиванием (`00:01:23 - 00:03:45`) или выбирайте отдельные **Главы (Chapters)** напрямую из YouTube без необходимости качать видео целиком.
- **Продвинутое аудио**: конвертация в MP3 320k, 192k и т.д.
- **Свои настройки медиа**:
  - Вшивание обложек (thumbnail) напрямую в файл.
  - Вшивание метаданных (author, title) в mp3/mp4/mkv файлы.
  - Скачивание и встраивание субтитров на выбранном языке.
- **Перехват буфера обмена (Clipboard Watcher)**: как только вы копируете ссылку (например, YouTube, TikTok, Vimeo), приложение предлагает скачать видео!
- **Профили (Пресеты)**: настройте свои форматы и папки (например, "В машину", "Архив 4K", "На телефон").
- **Кастомные темы**: современный интерфейс с переключателем различных тем (Светлая, Тёмная, Matrix, Synthwave и др.), а также динамическими анимированными обоями.

## 🛠 Стек технологий (Tech Stack)

Проект изначально построен на стеке **Vite + React + Electron + TypeScript**.

### Frontend (Renderer)
- **React.js 18** + **Vite** — для молниеносной отрисовки и сборки.
- **TypeScript** — строгая типизация и предсказуемость кода.
- **Zustand** — легковесный стейт-менеджер (управление темами, настройками, очередью скачивания).
- **i18next** (`react-i18next`) — полная поддержка перевода интерфейса (на лету).
- **Custom CSS** — гибкий, красивый и чистый стиль без привязки к тяжелым UI-фреймворкам. 

### Backend (Main Process)
- **Electron.js** — кроссплатформенная десктоп-среда (macOS, Windows, Linux).
- **yt-dlp** — сверхмощный CLI-движок для работы с медиаконтентом.
- **FFmpeg** — для соединения видео и аудио (merging), встраивания метаданных (metadata/thumbnail) и обрезки файлов прямо во время скачивания.
- **electron-store** — для безопасного хранения персистентных настроек и пресетов пользователя.

## 💻 Установка и запуск для разработчиков (Development)

Приложение требует установленного [Node.js](https://nodejs.org/). Бинарные файлы `yt-dlp` и `ffmpeg` приложение скачает и обновит автоматически при запуске!

1. **Склонируйте репозиторий**:
   ```bash
   git clone git@gitlab.com:apps4615111/yd.git
   cd yd
   ```

2. **Установите зависимости**:
   ```bash
   npm install
   ```

3. **Запустите режим разработки (Development)**:
   Приложение откроется в Electron с Hot-Reload (горячей заменой).
   ```bash
   npm run dev
   ```

4. **Соберите бинарные файлы для продакшена (Build)**:
   ```bash
   # Для Windows (.exe)
   npm run build:win

   # Для macOS (.dmg)
   npm run build:mac

   # Для Linux (.deb / .AppImage)
   npm run build:linux
   ```
