Выберите тему и нажмите «Скопировать CSS». Затем вставьте скопированный блок вместо блока :root { ... } в файле main.css.

Как применить:
1. Нажмите «📋 Скопировать CSS» под понравившейся темой
2. Откройте файл src/renderer/src/assets/main.css
3. Замените блок :root { ... } (строки 11–51) скопированным кодом
4. Vite автоматически применит изменения через Hot Reload
